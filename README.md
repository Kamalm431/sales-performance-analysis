# Sales Performance Analysis

## Overview
This project analyzes sales data using Python and SQL concepts to identify revenue trends and top-performing products.

## Tools Used
- Python (pandas, matplotlib)
- SQL concepts (GROUP BY, aggregation)

## Key Insights
- Identified top-performing products by revenue
- Analyzed monthly sales trends
- Calculated total revenue

## How to Run
1. Download the dataset
2. Run sales_analysis.py
